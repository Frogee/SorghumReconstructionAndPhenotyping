Computes point features for the input mesh.

Compute point features for use with machine learning. If the point is colored cyan (0, 255, 255), the point will be labeled as STEM, otherwise will be unlabeled. Features for all points are written to an output file (an .arf) that can be input to MultiBoost.

Estimate features. We use two "scales" of features to try and improve identification.

Without a doubt, the features calculated here can be (and should be) improved. Things to add: geodesic distance from bottom of plant? geodesic distance to center of plant?
