Detailed Description

Container to hold data corresponding to a graph that will be used for pathfinding via Dijkstra's algorithm.

The primary purpose is to keep track of which node label belongs to which index for the Dijkstra implementation we're using. The edges of the graph will be calculated using the point coordinates provided as part of _map_nodeLabelToPointCoordinate
